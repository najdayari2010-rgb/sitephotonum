Métadonnées EXIF

Le format EXIF (Exchangeable Image File) contient des informations techniques intégrées dans une photo numérique. Il enregistre des données comme la date, l'heure, le modèle de l'appareil et les paramètres du capteur. Il peut également inclure la localisation GPS si l'appareil est équipé. Ces informations permettent de retracer les conditions de prise de vue et d'organiser les photos. Le EXIF est très utile pour les photographes professionnels et amateurs qui souhaitent analyser ou classer leurs images.
